# 坑

## display 和 visibility

- display: none

元素被完全移除，不占据任何空间。
该元素及其子元素在页面上不可见，且不会影响布局。
例如，使用 display: none;的元素不会在 DOM 中占据任何位置。

- visibility: hidden

元素仍然存在于 DOM 中，但不可见。
该元素仍然占据空间，周围的元素不会重新布局。
例如，使用 visibility: hidden;的元素会在页面上保留其原有的空间，但内容不可见。

## border

至少有这两个属性才能生效

```
border-bottom: 1px solid
```
